## Introduction

Fireside Fibers is a Pygame project that simulates the relaxing world of knitting stitches. Whether you're a knitting enthusiast or just looking for a calming game experience, Fireside Fibers has something for everyone.

## Features

- Zen Mode: Enjoy knitting stitches at your own pace.
- Colorwork Mode: Create colorful patterns by choosing different stitch colors.
- Time Attack Mode: Challenge yourself with a time limit and see how many stitches you can knit.

## Game Modes

### 1. Zen Mode
In Zen Mode, you can enjoy knitting stitches without any time pressure. Take your time, relax, and create beautiful patterns at your own pace.

### 2. Colorwork Mode
Colorwork Mode allows you to choose from different colors for your stitches. Experiment with various color combinations and create stunning visual patterns.

### 3. Time Attack Mode
Challenge your knitting skills in Time Attack Mode. You have 30 seconds to knit as many stitches as possible. See how fast and accurate you can be under time pressure!

## How to Play
Navigate through the game modes using the numeric keys (1, 2, 3) in the main menu.
Follow on-screen prompts and use the specified controls for each mode.

## Controls
### Zen Mode & Time Attack Mode:
Q: Knit a stitch to the left.
P: Knit a stitch to the right.
Space: Change the needle image.
=> Key sequence : Q < Space < P < Space
### Colorwork Mode:
Numeric keys (1, 2) to choose stitch colors.
=> Key sequence : Numeric keys < Space < P < Space
